# knowledges

C/C++ 与 Linux 系统编程学习笔记与示例代码仓库。

## 目录

| 目录 | 描述 |
|---|---|
| `cpp/` | C++ 语言特性示例：类型推导、移动语义、异常处理、内存管理、const 用法等 |
| `network/` | Linux 网络编程：Socket API、TCP/UDP 通信、基于 select 的多人聊天室、网络协议笔记 |
| `linux/` | Linux 系统编程：IPC（管道/FIFO/消息队列/共享内存/信号量/信号/Unix Socket）、文件描述符 |
| `boost/datetime/` | Boost Date_Time 库示例：日期操作、计算、格式化、区间、计时器等 |
| `shell/` | Shell 脚本基础：变量、条件判断、数学运算、日期处理等 |
| `other/` | 其他笔记与示例：Git、RAII、锁机制、信号处理等 |
| `mysql/` | MySQL 相关（待补充） |
